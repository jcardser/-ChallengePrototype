# -ChallengePrototype
Contexto
Estás desarrollando un sistema para crear personajes en un videojuego RPG. Crear personajes desde cero es costoso porque implica cargar texturas, calcular estadísticas base, y configurar habilidades. Necesitas implementar el patrón Prototype para clonar personajes pre-configurados y modificarlos según sea necesario.

Objetivos
Implementar el patrón Prototype para clonar personajes
Crear diferentes tipos de personajes (Guerrero, Mago, Arquero)
Asegurar que la clonación sea profunda (deep copy) para atributos complejos
Permitir personalización después de clonar
